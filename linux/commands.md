# Linux Commands Cheat Sheet

## Basic Navigation
- `ls` - list files
- `cd` - change directory
- `pwd` - print current path

## File Operations
- `cp source dest` - copy
- `mv source dest` - move/rename
- `rm file` - remove file
- `mkdir name` - create folder

## Permissions
- `chmod 755 file` - change permissions
- `chown user file` - change owner

## System Info
- `uname -a`  
- `df -h`  
- `free -m`
